# 1. 系统简介

本项目是一个大型分布式 C++ 软件系统。

主要特点：

* 开发语言：C++
* 通信协议：CORBA
* 操作系统：

  * Windows Client
  * Linux Server
* 数据库存储：MySql
* 运行时间：多年持续维护

系统主要用于地铁设备综合监控：

---

# 2. 系统总体结构

整个系统分为四层：

```text
+--------------------------------------------------+
|                  Windows Client                  |
+--------------------------------------------------+
                │
                │ CORBA
                ▼
+--------------------------------------------------+
|                 CORBA ORB                        |
+--------------------------------------------------+
                │
                ▼
+--------------------------------------------------+
|                 Server Layer                     |
|                                                  |
| AlarmAgent                                       |
| EventAgent                                       |
| DutyAgent                                        |
| TrainAgent                                       |
| PAAgent                                          |
| PIDSAgent                                        |
| ...                                              |
+--------------------------------------------------+
                │
                ▼
+--------------------------------------------------+
|                 Database                         |
+--------------------------------------------------+
```

所有客户端均通过 CORBA 调用后台服务。

后台服务之间如有通信，也使用 CORBA。
