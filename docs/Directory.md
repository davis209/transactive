# Project Directory Guide

本文档用于帮助 Codex 快速定位代码。

## transactive/app

Manager和Agent相关应用层代码位于此目录。
比如：
* PAAgent
* PAManager

---

## transactive/bus

所有子系统业务逻辑层相关代码位于此目录。

---

## transactive/core

底层基础功能相关代码位于此目录。
比如：
* 数据库访问
* 数据结构
* 线程库

---

## cots/

第三方库相关代码位于此目录。
比如：
* boost
* ace
* omniCorba
